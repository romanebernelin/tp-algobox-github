# TP AlgoBox – Collaboration GitHub

Ce dépôt contient des exercices réalisés sur AlgoBox dans le cadre d’un TP visant à découvrir l’utilisation de GitHub comme outil de travail collaboratif.

## Contenu du dépôt
- Exercices AlgoBox (.algo)
- Algorithmes simples permettant de tester le suivi des versions avec GitHub

## Objectif du TP
- Créer un dépôt GitHub
- Ajouter des fichiers AlgoBox
- Comprendre le principe de commit et de versionnage
